# NoirOS 0.7.0-alpha.1 (alpha)

Data: 2024-05-26  \
Canal: alpha  \
Commit base: 048b96f

## Destaques
- Shell NoirCLI segmentado em módulos independentes com registro dinâmico de comandos.
- Reorganização da árvore de fontes seguindo arquitetura modular (arch/, core/, drivers/, fs/, security/, shell/, util/).
- Driver de teclado com layouts desacoplados (US e BR-ABNT2) e comando `config-keyboard` preparado para novos layouts.
- Makefile agora detecta automaticamente novos arquivos `src/**/*.c|s|asm`, simplificando integrações futuras.

## Quebras e migrações
- Cabeçalhos foram movidos para subdiretórios específicos; atualize includes customizados para os novos caminhos (`include/core`, `include/fs`, etc.).
- Scripts ou componentes que chamavam diretamente `print-version` devem considerar o novo formato `NoirOS <versão> [<canal>]`.

## Checklist pós-atualização
1. Recompile o kernel (`make clean && make`).
2. Atualize qualquer extensão externa para os novos caminhos de include.
3. Valide o NoirCLI executando `print-version` e `print-envs` para confirmar canal e versão ativos.

## Próximos passos
- Finalizar segmentação do NoirFS em módulos menores para reduzir o tamanho do arquivo atual.
- Preparar build beta 0.6.3 com foco em estabilidade do VFS.
