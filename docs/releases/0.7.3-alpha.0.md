# NoirOS 0.7.3-alpha.0 (alpha)

Data: 2025-12-15  \
Canal: alpha  \
Commit base: pending

## Correcoes e melhorias
- Layout BR-ABNT2 corrigido: dead keys consistentes, tecla extra `/ ?` (scancodes 0x59/0x73) e keypad com `,` como separador decimal; testes de host cobrindo as variantes.
- Hotkey `F1` injeta `help-docs` direto no prompt do NoirCLI.
- Instalador pergunta o layout de teclado antes de qualquer senha, garantindo entrada correta durante a formatacao.
- Comandos `shutdown-reboot` e `shutdown-off` adicionados, sincronizando buffers antes de reiniciar/desligar.
- Boot do kernel agora recusa operar sem NoirFS cifrado montado, eliminando prompt inseguro em discos sem formato.
- Camada de criptografia validada: falha de inicializacao nao prossegue com volume claro.

## Checklist pos-update
1. `make clean && make` ou `make iso` para gerar a ISO do instalador.
2. `make disk-img` para um disco de desenvolvimento e `make run-disk`.
3. No instalador, selecionar `br-abnt2` e validar acentos (`´`, `` ` ``, `~`, `^`, `"`) e tecla `/ ?`.
4. Na CLI, usar `F1` para abrir `help-docs` e testar `shutdown-reboot` / `shutdown-off`.
5. Confirmar que o boot recusa operar sem senha NoirFS (sem prompt inseguro).
