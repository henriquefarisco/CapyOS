# CapyUI v1

Primeira colecao visual oficial do CapyOS. Estes PNGs foram repetidos
byte a byte nas pastas de release `0.8.0-alpha.1` a `0.8.0-alpha.4`;
por isso foram consolidados em uma unica versao de interface.

Use esta pasta como baseline visual ate a captura oficial da proxima
versao de CapyUI.
