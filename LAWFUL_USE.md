# Lawful use statement

CapyOS is distributed under Apache-2.0 as Open Source software.

To remain Open Source, the repository license does not add a blanket
"no illegal use" restriction. Open Source licenses cannot impose that kind of
field-of-use limitation and still remain Open Source compatible.

Project stance:

- the project is intended for lawful research, education, hobbyist
  experimentation, and legitimate systems development
- the maintainer does not endorse unlawful use
- every user is solely responsible for complying with applicable law,
  regulation, sanctions, export controls, contracts, and third-party rights

This file is an informational statement only. It does not modify Apache-2.0
and does not create additional license terms.
